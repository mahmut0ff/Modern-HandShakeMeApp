# Chat UI Improvements

## Изменения

### 1. Исправлено отображение имен
- ✅ Вместо "User" теперь отображаются реальные имена пользователей
- ✅ Добавлен fallback на "Unknown User" если имя отсутствует
- ✅ Исправлено в списке чатов и в комнате чата

### 2. Улучшен дизайн списка чатов
- ✅ Добавлены градиенты для аватаров без фото
- ✅ Добавлены тени для карточек чатов (iOS и Android)
- ✅ Улучшена типографика (жирность для непрочитанных)
- ✅ Увеличены аватары (50px → 56px)
- ✅ Добавлены скругленные углы для карточек
- ✅ Улучшен индикатор онлайн статуса
- ✅ Улучшен badge непрочитанных сообщений с градиентом
- ✅ Добавлен счетчик непрочитанных в подзаголовке

### 3. Переход в профиль мастера
- ✅ Клик на имя/аватар в чате открывает профиль мастера
- ✅ Передается userId в ChatHeader
- ✅ Работает через onProfilePress и напрямую через userId

### 4. Рабочее меню с тремя точками
- ✅ Добавлено модальное меню с опциями:
  - View Profile - переход в профиль
  - Mute Notifications - отключение уведомлений (заглушка)
  - Report - жалоба (заглушка)
  - Block User - блокировка пользователя (заглушка)
- ✅ Красивая анимация появления
- ✅ Закрытие по клику вне меню

### 5. Индикатор непрочитанных сообщений
- ✅ Уже был реализован, но улучшен:
  - Градиентный фон
  - Лучшая типографика
  - Показывает "99+" для больших чисел
  - Счетчик в подзаголовке списка чатов
